# Streamlit 데이터 대시보드 페이지 기획서

## 1. 문서 목적

- 단일 또는 다중 **데이터 대시보드**를 Streamlit으로 구현할 때의 범위·우선순위·화면 구조·기술 선택을 고정한다.
- 기존 레포 패턴(`st.set_page_config`, 사이드바 필터, `st.cache_data`, Plotly 시각화)과 정렬한다.

## 2. 배경 및 목표

| 구분 | 내용 |
|------|------|
| **문제** | 원시 데이터·API 결과를 빠르게 탐색·비교·공유할 수 있는 화면이 필요함 |
| **목표** | 필터 → 요약 지표 → 차트 → 상세 표의 흐름으로 **의사결정에 쓰이는 정보**를 한 화면(또는 탭)에서 제공 |
| **비목표(초기)** | 로그인·권한, 실시간 초저지연 트레이딩, 대용량 ETL 파이프라인 |

## 3. 사용자·시나리오

- **분석/기획**: 기간·카테고리 필터로 지표를 바꾸며 트렌드를 본다.
- **운영/모니터링**: 최신 스냅샷·TTL 캐시로 API 부하를 줄이며 주기적으로 갱신한다.
- **발표/공유**: `wide` 레이아웃과 일관된 테마로 스크린 캡처·데모에 적합하게 만든다.

## 4. 정보 구조(IA)

```
대시보드 루트
├── 헤더: 제목, 캡션(데이터 출처·갱신 주기)
├── 사이드바: 필터(기간, 범주, 임계값 등)
├── 메인
│   ├── KPI 행 (st.metric / 요약 카드)
│   ├── 시각화 영역 (탭 또는 열 분할: 트렌드 / 분포 / 비교)
│   └── 데이터 테이블 (다운로드·정렬·페이지네이션 옵션)
└── 푸터(선택): 데이터 정의, 제한 사항, 링크
```

**다중 페이지**를 쓸 경우: `pages/` 디렉터리 또는 `st.navigation`으로 **데이터셋·도메인별** 페이지를 나눈다.

## 5. 기능 요구사항

### 5.1 필수

- **데이터 로딩**: 로컬 CSV 우선, 없으면 대체 소스(예: seaborn).
- **캐싱**: `@st.cache_data` + 필요 시 `ttl`로 API/IO 부하 제어.
- **필터**: 사이드바 위젯 — 필터 적용 후 모든 차트·표가 동일한 `filtered_df` 사용.
- **시각화**: Plotly.
- **접근성**: 컬럼 한글 라벨 맵(`COL_LABELS`)으로 표·범례 가독성 확보.

### 5.2 권장

- 빈 데이터·에러 시 `st.warning` / `st.info`.
- `st.download_button`으로 필터 적용 결과 CSV보내기.
- `st.expander`로 “지표 정의”·RAW 스키마 숨김.

### 5.3 선택(후순위)

- `st.session_state`로 필터 상태 유지.
- 인증·비공개 데이터는 별도 스코프에서 설계.

## 6. 비기능 요구사항

| 항목 | 지침 |
|------|------|
| **성능** | 필터는 메모리 내 DataFrame 연산 위주; 무거운 집계는 캐시 키 설계 |
| **신뢰성** | 외부 소스 실패 시 빈 프레임 처리·메시지 |
| **일관성** | `page_title`, `page_icon`, `layout="wide"`, `config.toml` 테마 통일 |
| **유지보수** | 데이터 로딩 / 전처리 / 차트 빌더 함수 분리 (`dashboard_io.py`, `dashboard_charts.py`) |

## 7. 기술 스택

- **UI**: Streamlit  
- **데이터**: pandas  
- **차트**: Plotly Express / graph_objects  
- **설정**: `.streamlit/config.toml`  
- **실행**: `streamlit run dashboard_app.py`

## 8. 구현 단계

1. **MVP**: 단일 데이터 소스, 사이드바 필터, KPI, 차트, 표, CSV 다운로드.  
2. **v1.1**: `pages/` 분리 또는 탭으로 도메인 확장.  

## 9. 코드 파일 매핑

| 파일 | 역할 |
|------|------|
| `dashboard_app.py` | 진입점, 레이아웃, 사이드바, 위젯 연결 |
| `dashboard_io.py` | 로딩·전처리·필터 순수 함수·`COL_LABELS` |
| `dashboard_charts.py` | Plotly Figure 빌더 |
